# LangChain Course

Questo progetto fa parte del corso su **LangChain** che sto seguendo per imparare a sviluppare applicazioni AI utilizzando il framework LangChain.

## Setup Ambiente Python

### Creare un Virtual Environment

Per creare un ambiente virtuale Python:

```powershell
python -m venv .venv
```

### Attivare il Virtual Environment

Per attivare l'ambiente virtuale su Windows PowerShell:

```powershell
.\.venv\Scripts\Activate.ps1
```

oppure:

```powershell
.venv\Scripts\activate
```

### Disattivare il Virtual Environment

Per disattivare l'ambiente virtuale:

```powershell
deactivate
```

### Installare le Dipendenze

Dopo aver attivato il virtual environment, installare le dipendenze dal file requirements.txt:

```powershell
pip install -r requirements.txt
```

Per verificare i pacchetti installati:

```powershell
pip list
```

## Note

- Assicurarsi di avere Python installato sul sistema
- L'ambiente virtuale aiuta a mantenere le dipendenze del progetto isolate
- Ricordarsi di attivare l'ambiente virtuale prima di installare pacchetti o eseguire script
